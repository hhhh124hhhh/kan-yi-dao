## ⚙️ 模型层新增字段（补全PRD对应逻辑）

| 变量            | 类型   | 说明               |
| ------------- | ---- | ---------------- |
| `location`    | str  | 当前训练场所（新手村、竹林等）  |
| `enemy_type`  | str  | 当前砍击对象（稻草人、竹人偶…） |
| `coins`       | int  | 铜币/资源，用于购买武器     |
| `inventory`   | list | 拥有的武器与物品         |
| `weapon_tier` | int  | 武器等级（对应刀光与数值提升）  |
| `ai_affinity` | int  | 与AI关系（决定AI语气变化）  |
